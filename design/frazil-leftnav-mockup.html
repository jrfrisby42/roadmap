Why this layout

Top bar: identity + global actions only (search, New item, account). It never changes between views, so nothing jumps. Filter row: owned by each view — shared chips first, "+ Filter" for the long tail, view-specific controls (Group by, Columns, Range) pinned right, and "Save filter" turns the current chip set into a one-click entry in the rail. Rail: Views with live counts, Projects, Saved filters, Settings/Admin at the bottom. Collapses to a 56px icon rail (default collapsed on Gantt) with [ as the shortcut.
